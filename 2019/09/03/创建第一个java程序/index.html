创建第一个java程序

  1. 创建一个文本文档HelloWorld.txt，然后更改txt后缀为java，用记事本打开，输入以下内容：

    1. 搜索cmd并打开

    2. 输入cd /d Directory（你存放Java程序的目录)

    3. 输入javac HelloWorld.java编译文件

      可以看见生成了一个class文件

    4. 输入java HelloWorld运行
